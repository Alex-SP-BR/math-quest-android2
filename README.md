# Math Quest

Aplicativo educacional desenvolvido em Kotlin para Android.

## 📱 Sobre o projeto

O Math Quest é um aplicativo focado na prática de matemática, com sistema de exercícios e progressão de dificuldade.

## 🚀 Funcionalidades

- Exercícios matemáticos
- Progressão de dificuldade
- Feedback ao usuário

## 🛠️ Tecnologias

- Kotlin
- Android Studio

## 📦 Publicação

Disponível na Google Play:
(https://play.google.com/store/apps/details?id=com.alex.mathquest)

## 🎯 Objetivo

Este projeto faz parte da minha evolução prática em desenvolvimento mobile.
